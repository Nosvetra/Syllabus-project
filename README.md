## How to Run Locally

### Prerequisite:

- Node.js must be installed

### Steps to Run:

1. Clone the repository:
   ```bash
   git clone https://github.com/Nosvetra/Syllabus-project.git
   ```
2. Navigate into Project folder
   ```bash
   cd Syllabus-project
   ```
3. Start the server
   ```bash
   node index.js
   ```
4. Open browser and visit
   ```bash
   http://localhost:3000/
   ```
